# Bitwise Subset Sum

### Problem Description

Given an integer $n$ and a sequence $a_1, a_2, \dots, a_n$. You need to process $q$ queries. For each query, you are given an integer $x$, and you must calculate the sum of $a_i$ for all indices $i$ that satisfy the following condition:

$$(x \ \& \ i) = i$$

**Terminology:** The symbol `&` denotes the **bitwise AND** operation. The condition $(x \ \& \ i) = i$ implies that $i$ is a bitwise subset of $x$ (i.e., every bit set in $i$ is also set in $x$).

---

### Input Format

The input contains multiple test cases. The first line contains an integer $T$ ($1 \le T \le 10^5$), the number of test cases.

For each test case:
1. The first line contains two integers $n$ and $q$ ($1 \le n, q \le 2 \times 10^5$).
2. The second line contains $n$ integers $a_1, a_2, \dots, a_n$ ($-10^9 \le a_i \le 10^9$).
3. The third line contains $q$ integers $x_1, x_2, \dots, x_q$ ($0 \le x_k \le 2 \times 10^5$).

**Constraints:**
- The sum of $n$ and the sum of $q$ over all test cases does not exceed $5 \times 10^5$.

---

### Output Format

For each test case, output $q$ lines. Each line should contain a single integer representing the answer to the corresponding query.

---

### Sample Input
```text
3
5 3
1 2 3 4 5
0 7 5
3 2
10 -5 7
3 2
6 3
0 1 2 3 4 5
6 4 7
```
### Sample Output
```text
0
15
10
12
-5
9
3
15
```
